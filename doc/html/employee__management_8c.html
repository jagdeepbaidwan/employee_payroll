Allows the admin to add a new employee in the database.

Allows the admin to add a new employee in emp_details and login_details in the database. This function is used either to add employee initially or in response to employee request by the manager. If responding to employee request by manager the input parameters will have auto-assigned values If adding employees initially the input parameter will have default values of "empty" for char type parameters and '0' for int type parameters and will be overwritten by user input
